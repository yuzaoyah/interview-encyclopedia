



## 347. 前 K 个高频元素

```python
class Solution(object):
    def topKFrequent(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: List[int]
        """
        counter = Counter(nums)
        keys = list(counter.keys())
        return sorted(keys, key=lambda x: -counter[x])[:k]
```
