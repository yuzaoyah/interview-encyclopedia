## 509. 斐波那契数

```
class Solution:
    def fib(self, n: int) -> int:
        if n < 2:
            return n
        f0, f1 = 0, 1
        for i in range(2, n + 1):
            fn = f0 + f1
            f0, f1 = f1, fn
        return fn
```
